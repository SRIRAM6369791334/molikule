# Molikule Enterprise Admin Dashboard - Complete User Guide

Welcome to the **Molikule Enterprise Admin Dashboard User Guide**. This document explains how to use each section of the admin panel step-by-step in simple English.

To help you navigate easily, **all sections in this guide use the exact names shown in your left sidebar menu.**

---

## Sidebar Navigation Menu Reference
Here is the mapping of sidebar names to their functions:
* **Dashboard**: Main stats screen.
* **Home Banners**: Manage the main sliders (1521x580px).
* **Banners**: Manage the homepage grid promo cards (Bento Cards).
* **Certificates**: Manage ISO and trust certifications.
* **Customers**: View website customer profiles (Read-Only).
* **Brands**: Register brand names and logos.
* **Categories**: Organize products into groupings.
* **All Products**: List, add, or bulk upload your products.
* **Product Variants**: Add size, volume, and scent variations.
* **Stock Management**: Manage product stock and inventory levels.
* **All Orders**: View every order placed on the website.
* **Pending Orders**: Orders awaiting verification.
* **Processing Orders**: Orders currently being packed.
* **In-Transit**: Shipped orders on their way to customers.
* **Delivered**: Successfully completed orders.
* **Discount Coupons**: Create promo discount codes.
* **Blog Manager**: Write and publish blog articles.
* **Product Reviews**: Approve or delete customer reviews.
* **Contact Messages**: Read messages sent via the contact form.
* **NEXUS Enquiries**: View NEXUS certification requests.
* **Job Applications**: View candidate resumes and details.
* **Open Positions**: Post hiring vacancies.

## 1. Login and Dashboard Overview

### How to Log In
1. Open your web browser and go to the admin login page (e.g., `yourdomain.com/login`).
2. Enter your registered **Email Address** and **Password**.
3. Click the **Log In** button.
4. If successful, you will be redirected to the main Dashboard page.
   * *Note: Only accounts with Admin privileges can log in.*

### Dashboard Overview
* **Stats Cards**: Displays quick counts of total orders, pending orders, processing orders, and today's orders.
* **Navigation Bar**: Located on the left side of the screen. Use this menu to navigate to different sections of the admin panel.

---

## 2. Interactive Tables (Grid.js)

The dashboard uses modern interactive tables (**Grid.js**) to list brands, categories, products, and orders.

### Key Table Features
* **Search Field**: Type keywords in the search bar to filter rows instantly.
* **Result Filters**: Drop-down menus to filter items by Brand, Category, or Status.
* **Clear Filter Button**: Click to reset the search instantly.
* **Pagination**: Click the page numbers at the bottom of the table to navigate through items.
* **Sorting**: Click any column header to sort the table rows in ascending or descending order.

---

## 3. Global Search and Suggestions

Located at the top header of the admin panel is the **Global Search Bar**. This is a powerful, instant lookup tool.

### How to Search
1. Click on the search bar at the top header of the screen.
2. Start typing a keyword (e.g., a product name, category title, or brand name).
3. Review the instant drop-down list of suggestions matching your keyword.
4. Click on any item in the suggestion list to navigate directly to its edit form.

## 4. Home Banners

Use this section to manage the slide banners that display on the main page of your website.

> [!IMPORTANT]
> **Banner Limit**: You can have a maximum of **6 active banners** at any time. If you already have 6 banners, the **Create Banner** button is disabled. You must delete an existing banner before creating a new one.

### How to Add (Create) a Banner
1. Click **Home Banners** in the left sidebar menu.
2. Click the **Create New Banner** button at the top right of the page.
3. Upload the **Banner Image** (Must be exactly **1521 x 580 pixels** and under **5 MB**).
4. Click the **Create Banner** button.
5. If successful, a green success message will appear, and the banner will go live.

### How to Edit a Banner
1. Go to the **Home Banners** list page.
2. Locate the banner you want to change and click the **Edit** (pencil icon) button.
3. Upload a new image in the upload area.
4. Click **Update Banner**.

### How to Toggle Status (Activate / Inactivate)
1. On the **Home Banners** list, locate the toggle switch next to each banner.
2. Click the switch to toggle between **Active** (visible on website) and **Inactive** (hidden).

### How to Delete a Banner
1. On the **Home Banners** list, click the **Delete** (trash icon) button next to the banner.
2. Confirm the delete request in the popup.

## 8. Brands

Use this section to manage the manufacturer names or brand labels.

### How to Add (Register) a Brand
1. Click **Brands** in the left sidebar menu.
2. Click the **Add Brand** button at the top right of the page.
3. Enter the **Brand Name**.
4. Upload the **Brand Logo** (Must be exactly **1024 x 1024 pixels** and under **5 MB**).
5. Switch the toggle to active so products under this brand can be displayed.
6. Check **Featured Brand** if you want to highlight this brand on the homepage.
7. Click **Register Brand**.

### How to Edit a Brand
1. Go to the **Brands** list page.
2. Locate the brand you want to change and click the **Edit** (pencil icon) button.
3. Modify the brand name, logo, or settings.
4. Click **Update Brand**.

### How to Delete a Brand
1. On the **Brands** list, click the **Delete** (trash icon) button next to the brand.
2. Confirm the delete request in the popup.
   * *Note: The system will block deletion if any active product is currently assigned to this brand.*

### Brand Bulk Upload CSV Format
1. Click **Bulk Upload** and click **Download Template** (`brands_template.csv`).
2. Fill in the column:

| CSV Column Header | Description | Example Value |
| :--- | :--- | :--- |
| `brand_name` | Name of the brand | *Molikule Pro* |

3. Save and upload.

## 10. All Products

Use this section to list new products and upload products in bulk.

### How to Add (Launch) a Product
1. Click **All Products** in the left sidebar menu.
2. Click the **Launch New Product** button at the top right of the page.
3. **Step 1: General Details**: Enter the name, select category/brand, upload the main image (Must be exactly **1080 x 1080 pixels**), upload gallery images, and enter descriptions. Click **Next: Inventory**.
4. **Step 2: Base Inventory & Pricing**: Enter flavor/scent, size/volume, MRP, discount values, initial stock quantity, and unique SKU. Click **Next: Shipping**.
5. **Step 3: Shipping & Status**: Enter package weight, length, width, height dimensions, set status to Active, and toggle Popular Pick if desired.
6. Click **Submit**.

### How to Edit a Product
1. Go to the **All Products** list page.
2. Locate the product you want to change and click the **Edit** (pencil icon) button.
3. Modify any of the fields in the multi-step form.
4. Click **Submit**.

### How to Delete a Product
1. On the **All Products** list, click the **Delete** (trash icon) button next to the product.
2. Confirm the delete request in the popup.

### Product Bulk Upload CSV Format
1. Click **Bulk Upload** and click **Download Template** (`products_template.csv`).
2. Fill in the template using the exact column headers below:

| CSV Column Header | Description | Example Value |
| :--- | :--- | :--- |
| `name` | Name of the product | *Wax polish* |
| `category_id` | Database ID of the Category | *1* |
| `brand_id` | Database ID of the Brand | *1* |
| `short_description` | Short summary line | *Wax Polish enhances shine.* |
| `description` | Full usage details | *Wax Polish is a protective...* |
| `weight` | Package weight value | *5.25* |
| `weight_unit` | Unit: `kg` or `g` | *kg* |
| `length` | Package length | *20.00* |
| `width` | Package width | *14.00* |
| `height` | Package height | *31.98* |
| `dimension_unit` | Unit: `cm` or `mm` | *cm* |
| `variant_value` | Scent/Flavor of first variant | *Wax Polish Scent* |
| `variant_unit` | Size/Volume of first variant | *5L* |
| `variant_mrp` | Original MRP price | *2400.00* |
| `variant_stock` | Initial stock quantity | *50* |
| `variant_sku` | Unique SKU code | *A-WX-01* |

3. Save and upload.

## 13. Order Management

Use these sections in the sidebar (**Pending Orders**, **Processing Orders**, **In-Transit**, **Delivered**, and **All Orders**) to track and process customer purchases.

### How to Update Order Status
1. Click any orders page (e.g., **Pending Orders** or **All Orders**) in the left sidebar menu.
2. Locate the order you want to update and click its details page.
3. Use the action dropdown to transition the order:
   * Select **Processing** when you begin packing. (Triggers an email notification).
   * Select **Dispatched** when handing over to the courier. (Triggers an email notification; order moves to **In-Transit**).
   * Select **Delivered** when the customer receives it. (Triggers an email notification; order moves to **Delivered**).
   * Select **Cancelled** to cancel the order and return stock back to inventory.

### How to View, Print, or Download an Invoice
1. Go to any order details page.
2. Click the **Invoice** action button:
   * Click **View** to see the invoice preview.
   * Click **Print** to send the invoice to your printer.
   * Click **Download** to save the invoice as a PDF file.
